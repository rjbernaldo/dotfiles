Channel Annotations Audit
As of May 2nd 2017, YouTube is no longer allowing you to Add or Edit Annotations.
We put this feature together to help you identify which of your videos still have annotations.

The "CSV" file that is generated can be imported into a Google Doc for you to reference as you update videos with Cards/End Screens.
Now that you have a list of Annotations across all your videos, you can manually start converting them into Cards / End Screens.

Or better yet, use our Bulk Copy Cards or Bulk Copy End Screens tools to complete the task in a fraction of the time :)
