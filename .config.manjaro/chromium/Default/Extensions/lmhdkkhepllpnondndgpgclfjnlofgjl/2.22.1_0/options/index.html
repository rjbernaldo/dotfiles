Support The Toolkit!


Thank you for installing Toolkit For YNAB!


This toolkit is completely free and is brought to you by the Open Source community

Issues or Feature Requests?


If you run into any issues or have ideas to make the Toolkit better in any way, submit an issue on github.

Want to track our progress?


Our complete road map is visible on our public Trello Board. Note that not all feature progress is tracked here but it does give the developers a good sense of what the community would like to see next.

How To Help


If you're a developer, hop on our Trello Board or look at our issue list on GitHub, find something that interests you and start coding away! Make a pull request when you've completed your feature and we'll get it into the next release as soon as everything looks good!

If you're not a developer, you can still help us tremendously by voting on cards listed in our Trello Board!

Thanks again for installing the Toolkit and Happy YNABing!
